# Smriti Sharma — Researcher Portfolio Page

This is a static GitHub Pages portfolio built with HTML, CSS, and JavaScript.

## What is included

- Interactive hero section with animated research-network background
- Thesis section with animated impact metrics
- Clickable architecture explorer for the memory-augmented web-agent system
- Project cards with filters
- Experience timeline
- Dark/light mode toggle
- Contact section

## How to customize

1. Open `index.html`.
2. Replace `your.email@example.com` with your real email.
3. Replace `https://www.linkedin.com/in/your-linkedin/` with your LinkedIn URL.
4. Put your CV in `assets/Smriti-Sharma-CV.pdf`, or update the CV link.
5. Add links to your thesis, GitHub repositories, publications, or demo videos in the project cards.

## How to publish on GitHub Pages

Option A: user site

1. Create a repository named `smritish21.github.io`.
2. Upload these files to the root of the repository.
3. Go to Settings → Pages.
4. Select the `main` branch and root folder `/` as the publishing source.
5. Your site should become available at `https://smritish21.github.io`.

Option B: project site

1. Create a normal repository, for example `research-portfolio`.
2. Upload these files.
3. Go to Settings → Pages and publish from `main` and `/`.
4. The site URL will look like `https://smritish21.github.io/research-portfolio/`.
