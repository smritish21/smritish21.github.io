Research direction

From one-off reasoning to reusable workflows

My research explores how web agents can reuse memory from previous tasks instead of reasoning from scratch every time. This matters for interactive web automation, lower compute cost, and more stable task completion.

01

Agent Memory

Storing task workflows, actions, thoughts, URLs, and embeddings so similar future tasks can be retrieved and reused.

02

Retrieval + Reranking

Using sentence embeddings and similarity search to surface useful prior experiences, then reranking them for context quality.

03

Reliable Evaluation

Measuring task success, step quality, efficiency, and near-success behavior to understand when memory actually helps.
